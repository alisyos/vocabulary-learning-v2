1) 지문 생성
2) 어휘 문제 생성
3) 문단 문제 생성
4) 종합 문제 생성
5) 과목
6) 영역
7) 구분(학습단계)
** 1~4번은 GPT 통신을 진행하는 단계별로 나누었습니다. 5~7번은 1~4에 동일하게 들어가는 변수 프롬프트 입니다. 그래서 별도로 구분하였습니다.

2. 탭 별 아래와 같이 구분해주세요.
1) 지문 생성
 - 전체 시스템 프롬프트
 - 지문 길이 별 프롬프트 (1-2문장으로 구성한 10개 단락, 1-2문장으로 구성한 12개 단락, 10문장 이하로 구성한 5개 단락, 4-5문장으로 구성한 5-6개 단락, 5-6문장으로 구성한 6개 단락)
 - 유형 별 프롬프트 (논설문, 탐구문, 사례지문, 인터뷰형지문, 비교/대조형지문, 실험/조사보고문)

2) 어휘 문제 생성
 - 전체 시스템 프롬프트
 - 문제 유형 별 프롬프트(출력 형식 포함): 객관식

3) 문단 문제 생성
 - 전체 시스템 프롬프트
 - 문제 유형 별 프롬프트(출력 형식 포함): 어절 순서 맞추기, 빈칸 채우기, 유의어 고르기, 반의어 고르기, 문단 요약

4) 종합 문제 생성
 - 전체 시스템 프롬프트
- 문제 유형 별 프롬프트(출력 형식 포함): 단답형, 문단별 순서 맞추기, 핵심 내용 요약, 핵심어/핵심문장 찾기

5) 과목: 과학, 사회

6) 영역: 지리, 일반사회, 정치, 경제, 화학, 물리, 생명, 지구과학

7) 구분(학습 단계): 중학생(1~3학년), 초등학교 고학년(5~6학년), 초등학교 중학년(3~4학년)
